Part 1: Quick Start and Interface

How to Enter Tags (The Main Rule)

Separate your tags with a comma. The application will automatically trim extra spaces. For example: 1girl, long hair, blue eyes.

Managing Tag Weights (‚ñ≤ / ‚ñº)

This is a key feature for controlling emphasis. Place your cursor right after the desired tag (or select it) and use the buttons to increase ((tag:1.1)) or decrease ((tag:0.9)) its impact.

Automatic Quality Tags

The "Add quality tags" option is enabled by default and automatically adds proven tags (masterpiece, best quality...) to the beginning of your prompt for maximum results. It is recommended to always keep this enabled.

Contextual Help (i)

Next to each section header, you will find an (i) icon. Hover over it to get a brief tip on what kind of information is best to enter in that field.

Preset Management: Import & Export

You manage your presets using .json files. Save successful combinations to the session, then export them to a file for permanent storage and sharing.

Part 3: Professional Tools

Using the Tag Browser (üìñ)

For technical fields like "Lighting" and "Camera," a special tool is available: the Tag Browser. Click the üìñ icon to open it.

  1. Select a subcategory on the left (e.g., "Shot Type" or "Light Source").
  2. Use the search bar for quick filtering.
  3. Click on tags on the right to select one or more (multi-select).
  4. Click "Add Selected" to add them to the text field.

Pro Tip: The Browser is the best way to discover new, powerful tags and understand how they work.

Tag Order Matters!

The neural network pays more attention to the first tags. A professional prompt is structured like this:

  1. Quality: masterpiece, best quality... (The app does this for you!)
  2. Technical Parameters: Light and camera. cinematic lighting, full body shot...
  3. Style: concept art, by greg rutkowski...
  4. Main Subject & Description: 1girl, long hair, in a forest...
